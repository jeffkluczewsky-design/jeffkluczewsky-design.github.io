# jeffkluczewsky-design.github.io

Personal GitHub Pages site for **Piotr Kluczewsky** — presenting the
**Agentic OS: Sales & Auditor Architecture** built on the
*Minimal Deviation Framework*.

Live site: https://jeffkluczewsky-design.github.io

Source project (code, diagrams, license):
https://github.com/jeffkluczewsky-design/apache

---

## What this repository contains

- `index.html` — the landing page. Everything (HTML, CSS, typography) is in this one file.
  Diagrams are loaded directly from the `apache` repository, so you never have to
  duplicate the images.

## How it is published

GitHub Pages is enabled on this repository, branch `main`, folder `/ (root)`.
Any push to `main` redeploys the site automatically within ~1 minute.
